# zsh-kubectl

A zsh plugin for the [kubectl cli](https://github.com/kubernetes/kubectl).

## Install
Using the [:zap: Zap](https://www.zapzsh.org/) minimal zsh plugin manager, add the following to your `.zshrc`

```sh
# Install plugin
plug "chrishrb/zsh-kubectl"
```
